### **Step 5: Test the Implementation**

1. **Start your development server:**
   ```bash
   npm run dev
   ```

2. **Check for TypeScript errors** in the console

3. **Test agent selection** in your UI - all 4 agents should appear

4. **Start a test session** with each agent to verify:
   - Agent loads correctly
   - First message uses the new templates  
   - System prompts include HSFB protocols
   - Memory integration works

### **Step 6: Verify Enhanced Memory Integration**

Your enhanced memory system should now work with all agents:

1. **Test returning user sessions** - should get personalized greetings
2. **Check console logs** - should show verbal acknowledgments being generated
3. **Verify different agent personalities** in their greetings and responses

### **Rollback Plan (If Issues Occur)**

If something breaks, quickly restore your original setup:

1. **Revert agent-configs.ts** to your original version
2. **Delete the new `/agents` folder**
3. **Restart development server**

### **Common Issues & Solutions**

**TypeScript Errors:**
- Make sure all imports in the new files are correct
- Check that file paths match exactly
- Verify no syntax errors in copied content

**Agents Not Appearing:**
- Check that `THERAPEUTIC_AGENTS` array exports correctly
- Verify the import in agent-configs.ts is correct

**Memory Integration Issues:**
- Ensure `firstMessageTemplate` functions are properly defined
- Check that enhanced memory service is still being called

**HSFB Not Working:**
- Verify HSFB protocols are included in system prompts
- Check meta tag structure matches expected format
