## LIKELY CAUSES

Based on the symptoms, here are the most likely issues:

### **Cause 1: KB Fetch Blocking VAPI Start**
The KB retrieval is synchronous (`await fetch`) and might be:
- Taking too long
- Returning an error
- Breaking the promise chain

**Fix:** Make KB retrieval non-blocking:
```typescript
// Don't await - fire and forget
fetch('/api/kb/retrieve', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    userId: userId,
    cssStage: 'pointed_origin',
    baseline: true,
    keywords: [],
    agentName: selectedAgent.name
  })
}).then(response => response.json())
  .then(kbData => {
    if (kbData.protocols && kbData.protocols.length > 0) {
      // Too late to add to system prompt, but log for verification
      console.log(`📚 KB protocols available (${kbData.documentCount} documents)`);
    }
  })
  .catch(error => {
    console.warn('⚠️ KB retrieval failed:', error);
  });

// Continue with VAPI start immediately
```
