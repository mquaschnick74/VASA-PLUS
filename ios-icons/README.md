# iOS App Icons Needed

Place the following PNG files in this directory, then run the icon copy script.

Required sizes (all PNG, no transparency, square):
- icon-20.png (20x20)
- icon-29.png (29x29)
- icon-40.png (40x40)
- icon-58.png (58x58)
- icon-60.png (60x60)
- icon-76.png (76x76)
- icon-80.png (80x80)
- icon-87.png (87x87)
- icon-120.png (120x120)
- icon-152.png (152x152)
- icon-167.png (167x167)
- icon-180.png (180x180)
- icon-1024.png (1024x1024) - App Store icon

Tip: Create the 1024x1024 version first, then use a tool like
https://appicon.co to generate all sizes automatically.
